# Instruct-TTS Stabilizer

Project page and audio demos for the Interspeech 2026 paper:

**Stabilizing Instruction Supervision for Instruct-TTS via Controllable Diversification and Drift Filtering**

The public project page is designed to be served through GitHub Pages from `docs/`.

- Project page: https://piedpiperg.github.io/instruct-tts-stabilizer/
- Paper PDF: `docs/instruct-tts-stabilizer.pdf`
- Demo assets: `docs/demopage/`

The page compares synthesized speech across Base, SFT, VoxInstruct, and the full recipe under APS, DSD, and RP instruction settings.
